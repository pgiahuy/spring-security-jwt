# Spring Security JWT

A simple authentication API built with Spring Boot using Spring Security and JWT

## Overview

This project demonstrates how to implement authentication and authorization using:

- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- PostgreSQL

The application provides a REST API for user registration, login, and secure access to protected endpoints
